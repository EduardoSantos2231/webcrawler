# Webcrawler para plataformas de emprego
## v0.0.1

#### Do que se trata?
Um script originalmente escrito em Typescript para fazer busca de oportunidades de estágio e rodar diretamente no terminal, os dados obtidos podem ser usados para fornecer a algum bot de automação de candidaturas ou ainda servir para análise de dados e estudos relacionados à média salarial, ou algo do tipo

##### Por que usar? 
A ideia é tornar a busca por estágios menos difícil, reunindo requisições às diversas plataformas que possuem API's expostas em um só lugar, nessa versão atual apenas é possível fazer essa busca com parâmetros definidos, mas o objetivo final é tornar isso interativo.

##### Objetivos futuros:
Um dos principais objetivos é tornar esse script mais performático e de mais fácil uso, permitindo que o próprio usuário informe parâmetros (região, salário, tipo de vaga, plataformas, quantidade de páginas).
